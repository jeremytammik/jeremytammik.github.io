API - Application Programming Interface

Eine API oder Application Programming Interface bietet einen Zugang zu einer Software von einer externen Applikation via einer Programmiersprache. Damit kann die Software von unabhängigen Applikationsentwicklern oder von dem Kunden selbst angepaßt werden.

Man kann eine API vergleichen mit einem Script-Interface, wobei ein Script nur die Möglichkeit bietet, die Software anusteuern, und keine echte Programmierung für die Lösung komplexerer Aufgaben erlaubt.
